# Requirements

- node & npm
- docker & docker compose

# Dev

- clone this repo and run `cd questions-demo`
- for backend run `cd backend && npm i && npm run start`
- for frontend run `cd frontend && npm i && npm run dev`

# Prod

- clone this repo and run `cd questions-demo`
- run `docker compose up -d`
